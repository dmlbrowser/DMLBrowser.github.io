# 平凡也精彩1
　　平凡的人，纯洁的心。雨幕中，我看到了他……

　　雨幕中，我看到了他…… 他，一个穿着绿色工作服的人；他，一个不顾严寒酷暑，风吹雨打，都一定完成任务的人；他，一个整天都在拖着一个没了点的电动车而忙碌的人；他，一个平凡的人，一个不寻常的人，一个平凡而又不平凡的人。他，就是一个爱岗敬业的邮递员。

　　那是去年暑假中一个多雨的坏天气，我和妈妈住在乡下的姥姥家避暑。妈妈在前些日上网购物时给姥爷选中了一个半袖T恤，邮局昨天给打电话，说妈妈空邮的包裹到了，今天就来给送。可是等到了中午，也没有送到，姥爷说：“下这么大的雨，路又不好走，今天就别惦记了，明天再说吧！”妈妈答应了一声，便去帮姥姥洗菜了。

　　嘿!他们不相信，我可是总有一种预感，邮递员叔叔，他，一定会来！到了下午两点多钟，我们早已吃了午饭，准备午休。突然，我听到了一阵——“砰！砰！砰！”的敲门声，我兴奋至极，急忙换鞋去开门，可姥姥，姥爷却不让，怕把我淋坏了，可是我真的很想去开门，就对姥姥，姥爷说:“我得赶快去开门，咱们只淋这么一小会儿，人家敲门的人估计早已淋成‘落汤鸡’了。”于是，我便趁姥姥，姥爷还没反应过来，就飞奔出去开门了。结果让我又惊又喜，是邮递员叔叔！他把包裹上的一块“大布解开，递给了我，然后又让我签了字。我看着叔叔被淋得狼狈不堪的样子，站在那里不动了，叔叔以为我还有什么事，便问我：“小朋友，还有什么事吗？如果没什么事的话，就赶快回去吧，别让大雨把你淋坏了。”我实在是猜不透了，便问叔叔：“叔叔，您的雨披呢？您怎么不穿呀？”叔叔没有回答我，只是冲我笑了笑，然后就用他那块“大布”又包上了另一个包裹（也就是他最后一个包裹，我看得出来），我疑惑不解地拿着包裹向家里走去，雨好像越下越大了。我慢慢回忆起叔叔刚才用来包包裹的“大布”，那不就是他的雨披吗？我豁然开朗，于是，便赶紧拿出自己家的雨披，向外面跑去，可是，叔叔已经走远了。我望着叔叔的背影，心里有说不出的滋味。

　　啊！平凡其实也精彩，我们要用平凡的生命，创造出不平凡的人生。就像李商隐所说的：“春蚕到死丝方尽，蜡炬成灰泪始干。”我们要用有限的生命，投入到为人民服务当中去。
   
# 平凡也精彩2

　　平凡不是平庸。一个平凡的人，你也就应为他鼓掌。
   
   我每一天上学都望见一位穿着绿衣服，戴着绿帽子的交通协管员在我们校园门口的马路上，不管是严寒还是酷暑，不管刮风还是下雨，他都会每一天站在马路的一旁，手举一面小旗，胸前挂着一个哨子，维持着交通，护送同学过马路。

　　每一天早上交通高峰时，也是同学们上学的时间，而我们校园门口的马路上又没有红绿灯，每一天空上学、放学的学生都要穿过马路上校园、回家。

　　当路边有等候过马路的学生时，他就面孔严肃地挥舞着小旗，挡住过往的车辆，让同学们安全的过马路，当有同学一个人要过马路时，他一边会拉着他们的手，一边吹着哨、一边挥舞着旗帜护送他们。这样的场面，让我小小地感动了一把。

　　如今，协管员几乎随处可见，他们天天站在马路上风吹日晒，维持交通秩序。平日里，我们不会去关注他们长什么样儿，可能也没觉得他们多么重要，甚至有的人想违章过个马路的时候还嫌他们碍事。

　　但是仔细想想，他们天天站在马路上，吃着灰尘，顶着风雨烈日，无论严寒、不论酷暑，有的人所作所为已经超出了“协管员”的职责范围，为过往的人们做了更多更多。

　　平凡的人不用华丽的金钻妆扮自己，不用闪耀的珠光点缀自己。许许多多平凡的人在我们身边默默做着平凡的事，给我带来精彩的生活。

　　“叮咚、叮咚”！只见张叔叔手里拿着像包裹的一包东西按着门铃。门开了，走出来一位阿姨，不耐烦的说到，我都等那么久了才送来，嘴里还喋喋不休的嘟囔着……她拿着包裹就转身回屋，张叔叔面带微笑的说了句：“不好意思，请您把签收单签个字！”她撇了张叔叔一眼，签完字便扭头走了。

　　张叔叔额头上的汗珠浸湿了他的衣领，头发上的汗在阳光的照射下，闪耀着光芒。他的工作是多么的辛苦，岗位是多么平凡，但那张饱经风霜的脸颊上却始终挂着淡然的笑容。我坚信，再平凡的人只要是付出努力，就必须会有回报。

　　平凡的世界，平凡的我们。但平凡并不代表平庸，只要付出努力，精彩总有一天会发生在这些平凡人的身上。

　　我想去做什么事情，就去做什么事情。从小，我就喜欢音乐。长大一点了，爸爸妈妈就开始给我找老师，上课。无论上什么课，最开始都是最难的，但只要经过反复的练习，最终必须会有成就的。到此刻为止，葫芦丝、电子琴和吉他，我都渐渐接触到了。每当我有不顺心的事情时，我都愿意随意地弹几首曲子。音乐就像是我的一个好朋友，一个知己，她能够在我开心的时候和我分享快乐，在我难过的时候与我分担痛苦。因为有了音乐，我变得不那么孤单了。

　　我喜欢缝十字绣，尽管父母总是说不让我弄那种东西，还说缝那种东西，既浪费时间，又影响视力和颈椎。有这闲工夫，还不如看看书，写写字呢。可我也不明白原因是什么，我就是喜欢。也许是看到别人绣很搞笑吧!不同的是，我享受的是过程。针在我的控制下，在十字绣的布格上挥舞着，使这块不起眼的布转成一块秀丽的艺术品。

　　我从小学三年级开始接触英语时，我对英语产生了浓厚的兴趣。听着英语老师说出一口流利的英语，我就羡慕的不得了。老师也很快的发现了我的英语天分，给我封了一个“官”，但因为我的粗心，英语老师似乎不是很高兴。但我还是很热爱英语。

　　我把自己的感情融入到我所喜爱的事情中，我坚信我是独一无二的。正如世界上没有两片完全一样的叶子似的。即使别人会认为我很普通，只是喜欢大多数人都喜欢的一些事情，但我在做的时候，还是感到快乐、充实的。

# 阳光路上2

　　生在世上,不可能总是一帆风顺,磕磕碰碰是在所难免的,更重要的是我们要有一颗永不消沉的心.

　　南方的秋天与北方秋天很大的不同.入秋一段时间了,却也还像春一样的多姿,像夏一样蓬勃.红花有绿叶的陪衬,绿叶有红花的相依,百花相媲美,其秋色绝不亚于春夏.看了这秋的一切,让人不禁喜上眉梢.一切是那么完美,就像我的心情一样地轻松,就像我的心一样的坦然.秋阳暖暖地照在我的脸颊上,心暖融融的,秋风也没有往日的萧瑟难耐.

　　那天晚上，我还是照常地晚休后经过那条僻静的小路,自然而然的经过了那个花卉园.也许是因为晚上的温度比较冷,花卉里照满了灯火,安安静静地,让我似乎是听到了一些小昆虫们温声细语地聊着天的声音.花儿绿叶们在不安分地玩弄着,不肯在夜幕下安静静地生长.听!那一句句窃窃私语,是它们生长的声音吗?是它们成长的足迹吗?感受着这一幕幕,我陶醉了.即使是秋天了,却还是那么年轻有活力,看!那些植物们不正是活力的象征吗?

　　原来一切都是那么美,忙碌无为的我却被琐事消磨了那些欣赏美的心.回过头看看,其实很多美的事物,只是我们没有用心去体会.生活是美好的,就算是秋天也会有秋的一番特色.

　　生活可以让我们无言,请不要遗失属于我们那份心德,让我们走在阳光路上,让我们的心与之一样地阳光与活力.

　　阳光路上，洒下了我们的汗水，也铸就了我们的辉煌。

　　阳光路上，依然有我们的脚印。
  
# 阳光路上4

　　走过春和夏，秋日的庄稼告诉我什么是幸福；走过风和雨，灿烂的阳光告诉我什么是温暖；阳光是什么？阳光是温暖，阳光是希望，阳光是理想，阳光也是成功。

　　清晨的微风将我从睡梦中唤醒，清脆悦耳的鸟叫声吵醒了还没睡醒的太阳，太阳极不情愿地从东方的大海里爬上蔚蓝的天空，一束耀眼、诱人的阳光照进屋里，瞧，他在向我微笑，那儿有一束打在了树叶上，让那翠绿的树叶闪着诱人的光，那儿还有，那儿也有……一束束耀眼的光唤起了我心中的那个理想，那个希望，那个梦。

　　自从踏进了中学的大门，我心中的那个理想，便像一颗飘来的种子，不经意间落在了我的心中，生根，发芽，在我的努力下，它竟结出了果实，那是一个名叫梦想的果子，里面承载了我的理想，为了这个梦想，我也曾努力过，放弃过，伤心过，欢乐过，记得那一年，我努力地学习，但始终不见收获，收到的只有一次次老师的批评、家长的教诲、同学的嘲笑，我下决心放弃，却不舍得自己的理想就这样被我扼杀在摇篮里，不，决不能放弃，理想是阳光，我要坚持。顶着别人的嘲笑，我的心就像葡萄一样，酸的很。有时候觉得自己就像一颗矮小的草，无人知晓，也无人问询。

　　一天，一棵长在花丛里的草竟从缝隙中伸出一片嫩叶来获取阳光，多么蓬勃向上的小草小草啊！原来小草的梦想就是阳光，我也要做一棵“小草”，去追寻我的“阳光”。直到今天，我这棵“小草”依然还在追寻着“阳光”，而这背后的酸甜苦辣又有谁会知晓？

　　多少追梦的身影，奔跑着拥抱希望，奔跑着追逐阳光，阳光路上风光无限，阳光路上幸福成长，阳光路上前程宽广。让我们走在阳光路上，昂首阔步向前，去追寻自己的梦想。
  
# 阳光路上5

　　有一条路通往天堂，下面却是修罗地狱，它美好但危险，那就是阳光路—题记

　　阳光路上，信念坚定的人始终坦然行走于那条康庄大道之上，而彷徨的人却总似身处于迷宫之中，来来回回却始终处于原点，止步起点线，有时这些人中的某一部分人急功近利，想要寻找捷径，导致最终走上不归路。

　　这是一条没有捷径的路，只有坚定地向前行走，才能找到通往美好未来的路。行走在阳光路上，花儿为我开，小草对我微笑，小鸟为我歌唱，弄得我也想长出一对翅膀来，翱翔于那蓝天之下，将这天底下的美景尽收眼底。那里没有红绿灯，也没有电线杆，我自由自在地飞翔。偶尔路上遇到小鸟，我向它们打招呼，它们可能会惊讶得而忘记了飞翔，跌在云上，我微笑着继续我的行程，继续着我那快乐地天空之旅。阳光路上总是充满着众多的神秘与美好，那里四季如春，每天只有花开，却不曾见过花谢，那里整天生机勃勃，到处都有一股芳香，沁人心脾，使人如沐浴于花池之中，倍感精神！

　　阳光路上，我与伙伴们一起歌唱着最美妙的大自然的乐章，每个人的脸上都挂满了笑容，此时此刻的我们仿佛是天底下最快乐、最快乐的人儿呀，那时这世界上的一切美好仿佛我们都拥有了，我们感到自己是如此的幸运与快乐，能够踏上这条漫长而美好的阳光路啊，好想时间凝固在这个时候，那我们便能永永远远的拥有这份快乐与美好。

　　阳光路上，充满了美好，但这是一条路只适合拥有坚定的信念的人行走呀！

# 就这样被你征服

　　人类骄傲地自称为世上最伟大的生物，但总有一些时刻，一些不经意间发现的点滴感动，就能将强悍外饰包裹下的脆弱心灵轻而易举地，完全征服。

　　我，就这样被你征服……

　　　　　　角落花

　　这是一个无法拥有生命活力的角落，没有光照，没有水分，没有土壤。然而就在这里，突然显眼地蹦出了一点绿色，托举着一抹娇艳的明黄。我作为一位匆匆的过路者，在一个不经意的角发现了这里——就是这么一株真真实实的绿色，从灰暗的水泥墙缝中探了出来，根茎弯得奇怪，向有阳光洒落的那个庭院伸展。就像是挣扎了许久，才从阴暗狭窄的墙缝中逃了出来，却仍无法沐浴温和的暖阳，于是努力地向着十几米开外那个洒满阳光的庭院伸长，希望总有一天可以触及那梦寐以求的阳光。是啊，为了心中的那缕灿烂，可以在不可能的环境里生存，可以挣扎扭曲着向有光的地方生长，可以如此执着……虽然，自己根本不能再生长十几米；虽然，自己根本就不能拥有阳光。但，角落里的花，还是依然倔强、坚强而又执着地活着。

　　我，就这样被你征服了！

　　驻留许久，我默默地身继续赶路。角落花，依旧向阳……

　　　　　　寒冬蝶

　　冬日，我与同学散布在教室外的走廊上，哆哆嗦嗦地谈着天，一边大口吸着教室里没有的纯净空气，一边不断地搓着双手取暖。云朵像是固定凝结在了灰暗的天空中，就像一幅画，却没有一丝丝的活力地闯入。

　　突然，一只缓慢飞舞着的蝴蝶跃入了我的视线，在没有生机的死寂寒冬，一只蝴蝶，便可以使人产生兴趣。我的目光追随着它，它就那么缓慢地飞着，无惧刺骨的寒风，在空中优美的飞舞，划过一道道柔美的弧线，向我展示着它的美丽。

　　我不知道为何它会突然出现在寒冬里，也不知道为何它要无惧寒风飞翔着。我只知道，在如此残酷恶劣的环境中，总还会有生命在展现在自己勃发而美丽的生命力。顽强、勇敢地继续着。

　　我，就这样被你征服了！

　　上课了，我慢慢地走进教室。寒冬蝶，依然迎风飞翔……
  

　　神奇的大自然，你总能在不经意间，带给我感动，这些感动，是如此真实，如此深切，如此震撼。神奇、美丽的大自然，我就这样，被你征服。
